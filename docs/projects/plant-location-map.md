# Industrial Plant Location Mapping — Jammu & Kashmir

## Overview

An interactive web map visualizing the locations of industrial plants
across Jammu & Kashmir, built using Python, Leafmap, and Folium.
The map includes clickable markers with plant names and addresses
sourced from OSM and government datasets.

## Tools & Technologies

- **Python** — data processing and map generation
- **Leafmap / Folium** — interactive map creation
- **OpenStreetMap** — base map tiles
- **Leaflet.js** — web mapping library

## Key Features

- Interactive markers with popup information for each plant
- Plant name and address displayed on click
- Responsive web map embeddable in any browser
